# Todos

- todo-input : 투두 추가 
    - 투두 추가 Input
    - 전체 선택
- todo-list :  투두 내용
    - 체크박스, 제목     
       hover 했을때 삭제버튼 등장
    - 더블클릭 했을 때 수정 Input 으로 변경
- todo-status 
    - 투두 상태 ( 전체, 할일, 종료 )
    - 투두 갯수
    - 종료된 투두 삭제


```javascript
const todos = {
    input:'',
    items:[
        {
            idx:0,
            name:'제목', 
            isEdit:false,
            isChekced:false,
        }
    ],
    status:'all'
}
```
